## Overview

**Simple Digital** is a fictional brand created for this portfolio project. The site works as a blog and resource hub for small businesses that want better websites, improved SEO, and stronger local online visibility.

Every page is built with semantic HTML, proper heading hierarchy, unique meta data, and accessible markup — the same best practices used on professional production websites.

---

## Features

- **Responsive Design** — Looks great on desktop, tablet, and mobile devices.
- **SEO-Optimized** — Unique titles, meta descriptions, heading structure, and internal linking on every page.
- **Blog Layout** — Homepage with featured article cards, a blog listing page, and three full blog posts.
- **Contact Form** — Client-side form validation with inline error messages and a success state.
- **Mobile Navigation** — Hamburger menu with smooth open/close animation on small screens.
- **Accessible** — ARIA labels, accessible form fields, semantic elements, and readable contrast.
- **Clean Code** — Well-commented, beginner-friendly HTML, CSS, and JavaScript.
- **No Dependencies** — Built entirely with vanilla HTML, CSS, and JS. Uses Google Fonts for typography.

## How to Run Locally

1. **Clone or download** this repository to your computer.
2. Open the project folder.
3. Double-click `index.html` to open it in your web browser.

No build tools, servers, or installations are required. The site is entirely static and runs directly in any modern browser.

Alternatively, you can use a local development server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then open `http://localhost:8000` in your browser.
